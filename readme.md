# Название пока не придумал

## О проекте:
2D аркада, цель в которой - пережить ночь в лесу, поддерживая огонь в костре.

В костер необходимо приносить ветки, которое будет разбросано по карте, либо срубать деревья с карты.

Так же по карте будут бегать контрабандисты, которые будут забирать дерево себе. Их можно будет догнать и забрать ресурсы себе.

С каждым уровнем количество деревьев будет уменьшаться, а скорость бандитов увеличиваться.
