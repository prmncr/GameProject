# Drobash Game

## Что это?
2D аркада с видом сверху про парня, который очутился в своем кошмаре - он попал в свой колледж, и теперь ему необходимо собирать процессоры и побеждать злых чудищ.

## Какие есть механики?
Главному герою будут противостоять несколько видов врагов (пока придумал только двух - один будет подбегать вплотную к персонажу, второй же будет стрелять) (даже реализовал, ого!).
Также запланирован магазин оружия, а именно - различных видов дробовиков, которые можно будет приобрести за игровую валюту - ваты (не те, которые мягкие и пушистые, а которые живут в электричестве) или из лутбоксов, которые случайно будут генерироваться на уровнях.
