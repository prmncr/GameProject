# Drobash Game
###### Финальная версия. К сожалению.

### Что это?
2D аркада с видом сверху про парня, который очутился в своем кошмаре - он попал в свой колледж, и теперь ему необходимо побеждать злых чудищ.

### Какие есть механики?
Главному герою будут противостоять несколько видов врагов (пока придумал только двух - один будет подбегать вплотную к персонажу, второй же будет стрелять).
Они немного умные (ну или глупые, на вкус и цвет как говорится), умеют бегать за игроком.
А впринципе и все. Еще можно рассказать про цель игры - зачистить уровень и найти выход.
